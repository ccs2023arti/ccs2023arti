User study

Collecting enviroment

Experiment setup:

  • Blink XT Camera
  • Cox network - Oklahoma
  • NETGEAR R6300 V2 Wireless AP (2.4 Ghz Mode Only)
  • Rooted Motolora Nexus 6
  • Kali Nethunter ROM 2021.1
  • Airmon-ng 1.6

Experiment

We recruited 5 volunteers and asked each of them to perform MotionCompass to figure out the location of a hidden wireless camera randomly selected and deployed in the aforementioned outdoor or indoor environment. Every participant performed 25 attempts for each environment.

Calculate method:

For Success Rate:
Success rate = Successful cases / Total cases
For detection time:
Detection Time = Time spent to get the camera status inferred

Data used:

We logged the following data for each count of camera status inferring try. For both Success rate and Detection Time data we use the result from the output of our app, the detection time data file for each individual tester is the output of app with csv format, then we manually log the error data from our app into matlab code.

Localization time Result

We output the total localization time into csv format (txt) files and then use python to do the data processing, here are the codes we used to generate the final result:

Calculate for user 1:

import  csv
from  statistics  import  mean
with  open('1.txt', newline='') as  f:
	reader = csv.reader(f)
	data = list(reader)
for  i  in  range(0, len(data[0])):
data[0][i] = int(data[0][i])
print("Min time of user 1 is ", min(data[0]))
print("Max time of user 1 is ", max(data[0]))
print("Average time of user 1 is ", str(mean(data[0])))
